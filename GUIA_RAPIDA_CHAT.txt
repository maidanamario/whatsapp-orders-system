🚀 PASOS PARA USAR:
═════════════════════════════════════════════════════════════════════════

1. EJECUTAR LA APLICACIÓN
   ───────────────────────
   
   Abre PowerShell/Terminal en el proyecto y ejecuta:
   
   python app.py
   
   Deberías ver:
   * Serving Flask app 'app'
   * Running on http://127.0.0.1:5000
   

2. ABRIR LA INTERFAZ WEB
   ──────────────────────
   
   Abre tu navegador y ve a:
   
   http://localhost:5000/chat
   
   O
   
   http://127.0.0.1:5000/chat


3. USAR EL CHAT
   ────────────
   
   a) Escribe tu primer mensaje: "hola"
   
   b) El sistema responderá con un menú:
      "1. Ver productos / 2. Hacer pedido"
   
   c) Elige una opción (escribe: 1 o 2)
   
   d) Sigue el flujo del chatbot


4. PARA SALIR
   ──────────
   
   Presiona Ctrl+C en la terminal

🎯 CARACTERES ESPECIALES / COMANDOS:
═════════════════════════════════════════════════════════════════════════

"hola"      → Inicia el chat y muestra menú
"0"         → Cancela el pedido actual
"si"        → Confirma un pedido
"no"        → Cancela un pedido
"menu"      → Vuelve al menú principal
"inicio"    → Vuelve al inicio

🔍 CARACTERÍSTICAS TÉCNICAS:
═════════════════════════════════════════════════════════════════════════

Usuario ID: Generado automáticamente
           ├─ Formato: web_user_XXXXX
           ├─ Donde XXXXX es un número aleatorio
           └─ Permite múltiples usuarios simultáneos

Validaciones:
           ├─ ✅ No permite mensajes vacíos
           ├─ ✅ Valida cantidad (ej: 2kg)
           ├─ ✅ Requiere usuario en JSON
           └─ ✅ Maneja errores de servidor

Persistencia:
           ├─ usuarios.json → Estado de cada usuario
           └─ pedidos.json → Órdenes confirmadas

Scroll:
           ├─ Automático al nuevo mensaje
           └─ Smooth (suave)

❌ ERRORES COMUNES Y SOLUCIONES:
═════════════════════════════════════════════════════════════════════════

PROBLEMA: "Cannot GET /chat"
SOLUCIÓN: Verifica que Flask esté ejecutándose en http://localhost:5000

PROBLEMA: Chat no responde
SOLUCIÓN: Abre la consola del navegador (F12) y mira si hay errores
         Verifica que el servidor esté activo

PROBLEMA: Mensajes no aparecen
SOLUCIÓN: Recarga la página (F5)
         Verifica JavaScript esté habilitado

PROBLEMA: "Mensaje vacío"
SOLUCIÓN: Debes escribir algo en el input antes de enviar
