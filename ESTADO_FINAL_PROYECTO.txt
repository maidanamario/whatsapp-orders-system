PARA EMPEZAR INMEDIATAMENTE:

1. Abre la terminal/PowerShell en la carpeta del proyecto

2. Ejecuta la aplicación:
   $ python app.py

3. Abre tu navegador en:
   http://localhost:5000/chat

4. Escribe "hola" en el chat

5. ¡Listo! Puedes empezar a usar el chatbot 🎉

                          📞 SOPORTE Y TROUBLESHOOTING

PROBLEMA: "Cannot GET /chat"
SOLUCIÓN: Verifica que Flask esté corriendo en http://localhost:5000

PROBLEMA: Chat no responde
SOLUCIÓN: Abre F12 → Console y mira si hay errores

PROBLEMA: Mensajes no aparecen
SOLUCIÓN: Recarga con F5, verifica JavaScript habilitado

PROBLEMA: Input no funciona
SOLUCIÓN: Verifica que messageInput tenga el ID correcto
