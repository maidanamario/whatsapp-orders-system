📱 CARACTERÍSTICAS TÉCNICAS
═══════════════════════════════════════════════════════════════════════════

USUARIO:
  - Generado automáticamente
  - Formato: web_user_[0-99999]
  - Almacenado en JSON (usuarios.json)
  - Permite múltiples usuarios simultáneos

CONEXIÓN:
  - Método: POST /mensaje
  - Payload: { "usuario": "web_user_123", "mensaje": "texto" }
  - Respuesta: { "respuesta": "respuesta_del_bot" }
  - Protocolo: JSON sobre HTTP

ESTADOS:
  - inicio
  - esperando_opcion
  - esperando_producto
  - esperando_cantidad
  - confirmando_pedido

PERSISTENCIA:
  - Usuario states: /data/usuarios.json
  - Pedidos: /data/pedidos.json

📝 EJEMPLOS DE CÓDIGO

CREAR USUARIO:
  const usuarioId = 'web_user_' + Math.floor(Math.random() * 100000);

ENVIAR MENSAJE:
  fetch('/mensaje', {
    method: 'POST',
    headers: { 'Content-Type': 'application/json' },
    body: JSON.stringify({ usuario: usuarioId, mensaje: texto })
  })

MOSTRAR MENSAJE USUARIO:

  - Burbuja verde (#25d366)
  - Alineada a derecha

MOSTRAR MENSAJE BOT:

  - Burbuja gris (#e0e0e0)
  - Alineada a izquierda
